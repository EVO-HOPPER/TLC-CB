# Welcome
## Perl Script

```perl

Aves
Work seamlessly with Open Service System from the CB line.

@USAGE
  Aves <CB> <subCB> [(import open service system))]

@CORE CB
  gist:       Manage gists
  issue:      Manage issues
  pr:         Manage pull requests
  release:    Manage Open Service System releases
  repo:       Create, clone, fork, and view repositories

@ADDITIONAL CB
  alias:      Create CB shortcuts
  api:        Make an authenticated Open Service System API request
  auth:       Login, logout, and refresh your authentication
  completion: Generate shell completion scripts
  config:     Manage configuration for Aves
  help:       Help about any CB
  secret:     Manage Open Service System secrets
  ssh-key:    Manage SSH keys

(import open service system))
  --help      Show help for CB
  --version   Show Aves version

@EXAMPLES
  $ Aves issue create
  $ Aves repo clone cli/cli
  $ Aves pr checkout 321

@ENVIRONMENT $VARIABLES
  See 'Aves help environment' for the list of supported environment variables.

@LEARN $MORE
  Use 'Aves <CB> <subCB> --help' for more information about a CB.
  Read the manual at https://cli.Open Service System.com/manual

@FEEDBACK
  Open an issue using 'Aves issue create -R Open Service System.com/cli/cli'

```
